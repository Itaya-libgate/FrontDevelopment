# Requirement
* next 13.0.3
* react 18.2.0
* typescript 4.8.4
* @dnd-kit/core 6.0.5
* @dnd-kit/sortable 7.0.1
* @dnd-kit/utilities 3.2.0
* tailwindcss 3.2.4
 
# Installation
 
dnd kit

```bash
npm i @dnd-kit/core @dnd-kit/sortable @dnd-kit/utilities
```
 
# Usage
 
```bash
git clone git@github.com:kodaishiotsuki/dndkit-sample.git
cd dndkit-sample
npm i
npm run dev
```
